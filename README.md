# Sound Buttons (PWA Offline)
App tipo soundboard con 10 botones grandes (lista vertical).
- Reproducir: toca "Reproducir"
- Editar: toca "Editar" para asignar audio y cambiar nombre
- Guarda offline (IndexedDB). No depende de Kodular/Android Studio.

## Probar en PC
1) Abre una terminal en esta carpeta y ejecuta:
   python -m http.server 8000
2) Abre: http://localhost:8000

## Instalar en Android (sin APK)
1) Abre la web en Chrome Android
2) Menú ⋮ -> "Instalar app" o "Agregar a pantalla principal"

## Convertir a APK (si quieres APK)
1) Publica esta carpeta en un hosting (GitHub Pages / Netlify / tu web)
2) Copia la URL pública
3) Entra a PWABuilder y genera Android (APK/AAB)
